# 포트폴리오 웹사이트

Next.js와 TypeScript로 제작한 개인 포트폴리오 웹사이트입니다.

## 기술 스택

- **프레임워크**: Next.js 15
- **언어**: TypeScript
- **스타일링**: Tailwind CSS
- **번들러**: Turbopack (개발 환경)

## 시작하기

개발 서버를 실행하려면:

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인하세요.

## 프로젝트 구조

```
portfolio/
├── app/
│   ├── globals.css
│   ├── layout.tsx
│   └── page.tsx
├── public/
├── package.json
└── README.md
```
